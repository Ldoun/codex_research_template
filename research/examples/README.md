# Golden Examples

These examples show the canonical artifact shapes that later skills and resume behavior expect.

- `scope-evidence-review/` shows the staged workflow after the background evidence-review phase and before experiment setup.
- `full-workflow/` shows the staged workflow through experiment setup, experiment logging, shared analysis updates, and final reporting.

All placeholder names such as `<approach-A>`, `<primary-metric>`, and `<seed-paper-A>` are illustrative only and should be replaced in real use.

Use these as references when checking whether a skill preserved the artifact contract.
