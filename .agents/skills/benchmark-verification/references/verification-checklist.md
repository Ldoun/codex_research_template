# Benchmark Verification Checklist

- What exactly is the claimed number or rank?
- Which source first published or currently hosts the claim?
- Is the dataset identical, including split/version/filtering?
- Is the metric identical?
- Is the evaluation harness or script known?
- Is prompting/few-shot/system setup known?
- Is this a paper result, repo result, model card result, or third-party result?
- Are code, config, or issue threads available to explain discrepancies?
- Can the comparison be called direct, only partial, or invalid?
