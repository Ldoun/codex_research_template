# Setup Checklist

Before any experiment work, confirm the minimum missing items:

- research goal
- deliverable type
- starting papers / repos / code paths
- primary metric and direction
- exact evaluation command
- current baseline
- fixed constraints
- minimum decision scale
- preferred approaches
- references to follow
- off-limits files
- compute budget
- approval boundaries

If one of these is unknown, make it visible in the brief instead of guessing.
