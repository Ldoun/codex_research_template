# Source Hierarchy for AI/ML Research

## Highest priority
1. Paper / technical report
2. Official repository
3. Model card or official documentation
4. Official benchmark documentation
5. Official issue thread or release note

## Secondary
6. Reputable third-party evaluation writeup with primary-source links
7. Blog post summarizing primary sources

## Lowest priority
8. Social posts, unsourced leaderboard screenshots, marketing pages without traceable evidence

## Rule
If a lower-tier source disagrees with a higher-tier source, preserve the conflict and prefer the higher-tier source unless there is clear newer evidence.
